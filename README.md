# CryptoVaultWallet
Software Engineering Year 2 Project, Group 35

## Tech Stack
- Frontend: HTML, CSS
- Backend: Node.js
- Database: MySQL

## Setup
1. Clone repo
2. Run `npm install`
3. Configure `.env`
4. Run `node app.js`
